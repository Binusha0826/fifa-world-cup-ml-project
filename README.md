# 🏆 My First Solo Machine Learning Project

This is my **first individual ML project**, done as personal practice. I wanted to understand the full 
ML workflow end-to-end on my own before diving into the real assignment.

## 🎯 Goal
Predict how FIFA World Cup finals are won (Regular Time / Extra Time / Penalties) 
using historical match data (1930-2022).

## 🔍 What I Learned
- How to load, clean, and encode real-world data
- How to train and evaluate a classification model
- **Most importantly**: why dataset size matters — with only 22 samples, 
  my model struggled to generalize, which taught me more about ML limitations 
  than a "perfect" result would have!

## 🛠️ Tools Used
Python, Pandas, Scikit-learn, Google Colab

## 📊 Result
Accuracy was low due to the small dataset — but that's the whole point of 
this exercise. Onward to a bigger dataset for the real coursework! 🚀
